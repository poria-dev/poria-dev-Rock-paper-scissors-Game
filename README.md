# 🎮 About The Project

Rock Paper Scissors is a classic game recreated with a modern design and smooth user experience.

This project was built completely with **Vanilla JavaScript**, without using any libraries or frameworks.

It demonstrates DOM manipulation, event handling, responsive design, and clean JavaScript architecture.
